== Description ==
The WooCommerce PayJunction gateway plugin allows you to accept online payments in your WordPress WooCommerce store. Download the WooCommerce Payjunction plugin and start accepting payments from all major brands with your PayJunction account: Visa, MasterCard, American Express, Discover and more.

<strong>WooCommerce plugin</strong><br>
WooCommerce plugin is an e-commerce toolkit that helps you sell anything. Transform your WordPress website into online eCommerce store. Delivering enterprise-level quality, features and add ons to help you control your users experience.

This plugin works with the WooCommerce plugin and allows you to accept payments via your online store.

== Installation ==
<strong>Simply follow these steps to install:</strong><br>
1. Unzip the file.<br>
2. Upload the "woocommerce-gateway-PayJunction" folder to your WordPress Plugins directory.<br>
3. Login to your WordPress Admin, then go to Plugins and activate the "WooCommerce PayJunction Gateway" plugin<br>
4. Within the WordPress Admin, go to WooCommerce >> Settings, then click on the Payment Gateways tab, then click on the PayJunction link.<br>
5. Enter the proper information...<br>

<strong>To use in "Live Mode"</strong>
	<br>
1. Gather your QuickLink API Login and Password as explained on the following article: https://company.payjunction.com/pages/viewpage.action?pageId=328435<br>
2. Login to your WordPress Admin.<br>
3. Go to WooCommerce >> Settings >> PayJunction.<br>
4. Un-check "Enable PayJunction Test".<br>
5. Enter your API Username (See step 1 above).<br>
6. Enter your API Password (See step 1 above).<br>
7. Click "Save". Your website/shopping cart is now setup to process transactions via your PayJunction acount. We recommend that you process a test transaction, then login to your PayJunction account to confirm that it appears in your Current Unsettled Batch (aka Transaction History).<br>
   
<strong> To use in "Test Mode" </strong><br>
1. Login to your WordPress Admin.<br>
2. Go to WooCommerce >> Settings >> PayJunction.<br>
3. Check the box for "Enable PayJunction Test".<br>
4. Use the following API Username = pj-ql-01<br>
5. Use the following API Password = pj-ql-01p<br>

== Frequently Asked Questions ==
<strong>Do I need a Merchant account before I can use the plugin to accept payments?</strong><br>
Yes. In order to use this plugin to accept transaction in live mode you will need to have a merchant account. For more information please visit : <a href="http://pricing.payjunction.com/woothemes" target="_blank"> <strong>Payjunction</strong></a> If you already have a merchant account set up with <a href="http://pricing.payjunction.com/woothemes" target="_blank"> <strong>Payjunction</strong></a> all you will need is your API username and password.<br>
<br>
<strong>Do I have the ability to place in a test mode or sandbox during development?</strong><br>
Yes. You would need to enable "PayJunction Test Mode" in the WooCommerce settings<br>
<br>
<strong>What is the cost of the plugin?</strong><br>
The plugin is a FREE download, however it does require that you have account with <a href="http://pricing.payjunction.com/woothemes" target="_blank"> <strong>Payjunction</strong></a> If you already have a merchant account set up with <a href="http://pricing.payjunction.com/woothemes" target="_blank"> <strong>Payjunction</strong></a> which has costs. To learn more about our pricing please visit : <a href="http://pricing.payjunction.com/woothemes" target="_blank"> <strong>Payjunction</strong></a><br>
<br>
<strong>PayJunction is not showing up in live mode.</strong><br>
For security purposes, this plugin requires that you have an active SSL connection via a secure https page to view this payment option on your payment pages.<br>
<br>
<strong> I am getting an API error</strong><br>
Double check your API username and password are entered correctly on settings page <br>
